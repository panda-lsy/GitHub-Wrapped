# GitHub Wrapped 🎁

**你的年度代码之旅，可视化呈现。**

GitHub Wrapped 是一款专为开发者打造的年度总结工具。它分析你过去一年的 GitHub 活动，通过沉浸式的滑动体验，生成一份精美、可分享的编程旅程总结。

[English](README.md) | [**🚀 立即体验**](https://githubwrapped-delta.vercel.app)

---

## ✨ 功能特性

### 🎨 沉浸式体验
- **交互式幻灯片**：通过流畅的动画和过渡效果浏览你的年度数据
- **键盘支持**：使用左右箭头键在幻灯片之间导航
- **进度指示器**：跟踪你在幻灯片中的进度

### 📊 全面统计分析
- **总贡献数**：提交代码、拉取请求、问题提交和代码审查的综合统计
- **获得的 Star 数**：所有仓库获得的 Star 总数
- **Pull Requests**：追踪你的 PR 贡献
- **创建的 Issue**：通过问题报告看到你的影响力
- **代码审查**：你审查的 PR 数量
- **最长连续贡献**：你最长连续贡献的天数
- **当前连续贡献**：你当前的连续贡献天数
- **最佳贡献日**：你贡献次数最多的一天

### 🗓️ 贡献热力图
- 通过精美的热力图可视化你每日的代码活跃度
- 查看贡献历史中的连续记录和模式
- 带有贡献级别的交互式日历视图

### 🏆 成就徽章
- **贡献徽章**：根据总贡献数解锁徽章（100、500、1K、2.5K）
- **连续贡献徽章**：通过保持连续贡献获得徽章（7、30、100 天）
- **Star 徽章**：为获得的 Star 收集徽章（10、50、100、500）
- **PR 徽章**：对拉取请求的认可（10、50、100）
- **特殊徽章**：多语言专家（5+ 种语言）、代码审查员（100+ 次审查）
- **进度追踪**：查看你接近锁定徽章的进度

### 🎯 关键里程碑
- **1K 贡献**：庆祝达到 1,000 次贡献
- **2.5K 贡献**：精英贡献者成就
- **100 Star**：对仓库受欢迎度的认可
- **30 天连续贡献**：对持续编码的承诺
- **10 个仓库**：活跃的仓库创建者

### 🌍 常用编程语言
- 编程语言的可视化分布
- 按语言百分比细分
- 带有颜色指示的语言标记
- 追踪你的语言多样性

### 📦 热门仓库
- 你最受欢迎的仓库
- 仓库描述信息
- 每个仓库的主要语言
- Star 数量和分支信息
- 开放的问题追踪

### 🔄 快捷操作
- **刷新数据**：使用最新的 GitHub 活动更新你的年度总结
- **社交分享**：直接分享到 Twitter/X 和 LinkedIn
- **下载图片**：将你的总结保存为图片（即将推出）
- **多语言支持**：在中英文之间切换

### 🔒 隐私优先
- **只读访问**：仅请求读取公开数据的权限
- **无数据存储**：你的数据永远不会存储在我们的服务器上
- **安全认证**：使用 NextAuth 进行 GitHub OAuth 认证

## 🚀 如何使用

1. **连接账号**：访问 [应用主页](https://githubwrapped-delta.vercel.app) 并使用 GitHub 账号登录
2. **探索数据**：通过交互式幻灯片浏览你的统计数据
   - 使用左右箭头键或点击导航按钮
   - 查看你的介绍、统计、热力图、语言、仓库、徽章和里程碑
3. **分享成就**：在社交平台上分享你的成就
   - 点击 Twitter 或 LinkedIn 按钮直接分享
   - 使用 `#GitHubWrapped` 加入社区
4. **刷新数据**：点击刷新按钮用最近的活动更新你的数据

## 🎯 幻灯片概览

1. **介绍页**：带有用户个人资料和简介的个性化欢迎
2. **统计页**：关键指标和影响力数字
3. **热力图页**：带有连续记录的可视化贡献日历
4. **语言页**：编程语言分布
5. **仓库页**：带有详情的热门仓库
6. **徽章页**：成就徽章（已解锁和锁定）
7. **里程碑页**：关键成就和里程碑
8. **总结页**：感谢和行动号召

## 🛠️ 开发者指南：部署你自己的版本

想要托管属于你自己的版本或参与开发？

### 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FFreakz3z%2FGitHub-Wrapped)

### 本地开发

1. **克隆仓库**：
   ```bash
   git clone https://github.com/Freakz3z/GitHub-Wrapped.git
   cd GitHub-Wrapped
   ```

2. **安装依赖**：
   ```bash
   npm install
   ```

3. **配置环境变量**：
   创建一个 `.env.local` 文件并填入以下内容：
   ```env
   GITHUB_ID=你的_github_client_id
   GITHUB_SECRET=你的_github_client_secret
   NEXTAUTH_SECRET=你的随机密钥
   NEXTAUTH_URL=http://localhost:3000
   ```

4. **运行应用**：
   ```bash
   npm run dev
   ```

### 技术栈

- **框架**：Next.js 16 with App Router
- **UI**：React 19, Tailwind CSS 4
- **认证**：NextAuth with GitHub OAuth
- **数据**：GitHub GraphQL API
- **动画**：Framer Motion
- **图表**：Recharts, react-activity-calendar
- **图标**：Lucide React
